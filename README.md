# Video Downloader Pro

مشروع Android أولي مكتوب بـ Kotlin وJetpack Compose لإدارة تنزيلات الوسائط المسموح بها، مع بنية قابلة للتوسع لإضافة التحليل، السجل، الجدولة، والعمل في الخلفية.

> **تنبيه قانوني:** استخدم التطبيق فقط لتنزيل محتوى تملكه أو لديك إذن صريح لتنزيله. يجب احترام حقوق النشر وشروط استخدام كل منصة، ولا يتضمن المشروع آليات لتجاوز DRM أو الحماية التقنية.

## الحالة

هذه نسخة MVP عملية تتضمن:

- تحليل روابط الوسائط عبر yt-dlp وعرض العنوان والصورة والمدة.
- تنزيل الفيديو مع اختيارات أفضل جودة و1080p و720p و480p.
- استخراج الصوت بصيغة MP3 عبر FFmpeg.
- عرض تقدم التنزيل والوقت التقديري.
- حفظ الملفات داخل `Downloads/VideoDownloaderPro`.
- Room وWorkManager كأساس للسجل والمهام المجدولة.
- Android 8.0+، وTarget SDK 34، وJDK 17.

## المتطلبات

- Android Studio Hedgehog أو أحدث
- JDK 17
- Android SDK 34

## التشغيل

1. افتح المجلد في Android Studio.
2. دع Gradle ينفذ المزامنة.
3. شغّل التطبيق على جهاز أو محاكي Android 8.0+.

## خارطة الطريق

- [ ] طبقة Repository وViewModel باستخدام StateFlow.
- [ ] تحليل روابط المصادر المسموح بها وعرض الصيغ.
- [ ] Foreground Service مع إشعارات التقدم.
- [ ] جدولة التنزيل بقيود Wi‑Fi والطاقة.
- [ ] سجل التنزيلات وإدارة الملفات عبر MediaStore/SAF.
- [ ] اختبارات وحدات وتكامل وواجهة.
- [ ] تعريب كامل وإتاحة الوصول.

## ملاحظات Android

- Android 13+ يتطلب طلب إذن الإشعارات وقت التشغيل.
- يفضّل MediaStore أو Storage Access Framework بدل أذونات التخزين القديمة.
- يجب تعريف نوع Foreground Service المناسب عند تنفيذ خدمة التنزيل.
